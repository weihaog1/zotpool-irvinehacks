# Code Style & Conventions

## TypeScript
- Strict TypeScript with explicit types
- Interfaces/types defined in `types.ts`
- React.FC for functional components

## React
- Functional components with hooks
- Single AppContext for global state
- HashRouter for routing (GitHub Pages compatible)

## Styling
- Tailwind CSS utility classes
- Custom UCI colors: `uci-blue`, `uci-gold`, `uci-dark`
- Glass morphism effects with backdrop-blur
- Responsive design with sm:/md:/lg: breakpoints

## File Organization
- Pages in `/pages/` directory
- Reusable components in `/components/`
- One component per file, named exports
